# botv3 CAD Model

This is the STEP file for the botv3 mobile robot assembly.

## Files

- `botv3_v5.step` — Complete CAD model with all components and assembly structure.

## Getting Dimensions in Fusion 360

1. Open `botv3_v5.stl` in Fusion 360
2. Go to **Inspect → Measure** (or press `M`)
3. Click on edges, faces, or bodies to get distances and dimensions
4. For overall bounding box: select the entire assembly and check the **Properties** panel on the right
5. Measurements are in **millimetres (mm)**
